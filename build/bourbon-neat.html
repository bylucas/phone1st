Make My Bourbon Neat

Bourbon, a lightweight library of sass mixins from thoughtbot, helps designers create quick, quality code. Being a small library it doesn't take long to pick-up. Neat is built on top of Bourbon, and is a lightweight grid framework for Sass. And like Bourbon, being lightweight it's easy to pick-up.

Bourbon

The Bourbon mixin library had quite a change for version 5, becoming a lot lighter. Now up-to version 6, not much different from 5, the mixins below are what's available at present (keep checking the Bourbon website for mixin information). For more information about the Bourbon mixin library see Bourbon on the Github website and Bourbon Docs on the Bourbon website.

Modular Scale

Increments up or down a defined scale and returns an adjusted value. This helps establish consistent measurements throughout your project.
For more information about Modular Scales start here

//Globally change the base ratio
$bourbon: (
"modular-scale-ratio": 1.2,
);
.element {
font-size: modular-scale(3);
}
//CSS Output
.element {
font-size: 1.728em;
}

This text is using the style above to set the font size

Overflow Wrap

Outputs the overflow-wrap property and its legacy name word-wrap to support browsers that do not yet use overflow-wrap

Accepted CSS values are normal, break-word, inherit, initial or unset.
@example scss
.wrapper {
@include overflow-wrap;
}
//CSS Output
.wrapper {
word-wrap: break-word;
overflow-wrap: break-word;
}

Neat Grids

Bourbon Neat, a lightweight grid framework for Sass, totally changed in version 2, now up-to version 6, it's easy to pick-up but unfortunately it's no longer maintained. Phone1st uses CodeKit's copy of the Neat library and shown here are some basic grid settings, but with Flexbox and CSS Grids are the grid framework's day's numbered?

Don't forget to check the Bourbon Neat website for more information.

Something to watch

Take a look at Page with one sidebar and Page with two sidebars. Both have a background color, red. Notice the Page with one sidebar has margins left and right set to whatever the gutter is set to. This seems to be default which seems strange. The Page with two sidebars has no margins left or right but has an extra div element set to collapse the grid, which works a treat. Also see the Bourbon Neat website for more examples and info.

Grid Variables
@import "neat";

//over-riding the default neat gutter setting
$neat-grid: (
  columns: 12,
  gutter: 10px,
);
//new grid settings
$small-grid: (
  columns: 4,
  gutter: 10px,
  media: 600px,
);
$medium-grid: (
  columns: 8,
  gutter: 20px,
  media: 740px,
);
$large-grid: (
  columns: 8,
  gutter: 20px,
  media: 900px,
);
$box-color: #486cb8;
$height: 120px;
